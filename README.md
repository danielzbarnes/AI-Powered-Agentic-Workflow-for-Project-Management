# AI-Powered-Agentic-Workflow-for-Project-Management

The goal of this project is to crate a reusable agentic workflow.

there are 2 phases.

In Phase 1, the agentic toolkit is defined within the `base_agents.py` which has severala agentic classes that are individually tested with stand alone scripts.  

In Phase 2, the `agentic_workflow.py` script orchestrates a slection of Phase 1 agents to performe multi-step technical project management tasks.
